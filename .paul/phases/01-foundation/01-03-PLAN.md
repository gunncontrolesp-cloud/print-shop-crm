<objective>
## Goal
Build the admin-editable pricing table (Settings screen) and the Quote Builder — giving the shop the ability to generate formula-priced quotes from a live pricing config with product types, quantity breaks, material multipliers, and finishing multipliers.

## Purpose
The pricing table and quote builder are the shop's core business tools — quoting is how every job starts. Without accurate pricing, every quote is manual guesswork. This plan replaces that spreadsheet with a formula-driven system the admin controls.

## Output
- `pricing_config` table (single-row JSON config) + `quotes` table with `line_items` JSONB
- Settings/Pricing screen at /dashboard/settings/pricing (admin-only)
- Quote list, create, and detail pages
- Interactive Client Component that calculates line item pricing in real time using the formula: `unit_price = base_price × qty_multiplier × material_multiplier × finishing_multiplier`
- Quotes nav item enabled in sidebar
</objective>

## AC-2: Quote Builder Calculates Correctly
```gherkin
Given a pricing config with known values
When a user builds a quote line item selecting product type, qty, material, and finishing
Then the unit price and line total update automatically using the formula:
  unit_price = base_price × qty_multiplier × material_multiplier × finishing_multiplier
  line_total = unit_price × qty
And multiple line items can be added, each calculated independently
And the quote total is the sum of all line item totals
```

And an admin can "Approve" or "Reject" a sent quote

    create policy "Authenticated users can view quotes"
      on public.quotes for select
      using (auth.uid() in (select id from public.users));

    -- Staff + admin can create quotes
    create policy "Staff can create quotes"
      on public.quotes for insert
