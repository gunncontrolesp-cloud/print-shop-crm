<objective>
## Goal
Let admins and managers send a one-click follow-up reminder email to customers who haven't responded to a sent quote. Track when the last reminder was sent to avoid spamming.

## Output
- `reminder_sent_at` column on quotes table
- "Send Reminder" button on quote detail (only visible when status = 'sent')
- `sendQuoteReminder(quoteId)` server action → fires n8n webhook → Resend email
- n8n webhook: `quote-reminder` event type (mirror of `quote-sent` flow)
- Button shows "Reminder sent [date]" after first send; "Resend Reminder" after 48h
</objective>

<acceptance_criteria>

## AC-1: Send Reminder button appears on sent quotes
```gherkin
Given a quote with status = 'sent'
When an admin/manager views the quote detail
Then they see a "Send Reminder" button
```

## AC-2: Reminder fires n8n webhook
```gherkin
Given an admin clicks Send Reminder
When the action completes
Then n8n receives a webhook with quote details and customer email
  and reminder_sent_at is set to now()
  and the button changes to "Reminder sent [date]"
```

## AC-3: Cannot spam — cooldown enforced
```gherkin
Given a reminder was sent within the last 48 hours
When an admin views the quote detail
Then the Send Reminder button is disabled with "Sent [X hours] ago" tooltip
```

## AC-4: Reminders only for pending/sent quotes
```gherkin
Given a quote with status = 'approved' or 'rejected'
When an admin views the quote detail
Then no Send Reminder button is shown
```

</acceptance_criteria>

    In quotes.ts, add:
    ```ts
    export async function sendQuoteReminder(id: string) {
      await requireElevated() // admin or manager
      const supabase = await createClient()

      const { data: quote } = await supabase
        .from('quotes')
        .select('id, status, reminder_sent_at, customers(email, name), subtotal')
        .eq('id', id)
        .single()

      if (!quote || !['pending', 'sent'].includes(quote.status)) {
        throw new Error('Cannot send reminder for this quote')
      }

      // 48h cooldown
      if (quote.reminder_sent_at) {
        const hoursSince = (Date.now() - new Date(quote.reminder_sent_at).getTime()) / 3600000
        if (hoursSince < 48) throw new Error('Reminder sent too recently')
      }

      await notifyQuoteReminder(quote)

      await supabase.from('quotes').update({ reminder_sent_at: new Date().toISOString() }).eq('id', id)
      revalidatePath(`/dashboard/quotes/${id}`)
    }
    ```

    In n8n.ts, add `notifyQuoteReminder(quote)` following the same pattern as `notifyQuoteSent` — POST to QUOTE_REMINDER_WEBHOOK_URL env variable.
