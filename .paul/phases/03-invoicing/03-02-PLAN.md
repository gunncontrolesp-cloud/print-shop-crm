<objective>
## Goal
Add Stripe payment link generation to invoices — admin clicks "Send Payment Link", Stripe creates a hosted payment page, the URL is stored on the invoice, and the customer receives an email with the link via n8n. Built behind a PaymentProvider abstraction so Stripe can be swapped for PayPal or Clover without restructuring.

## Purpose
This completes the "order-to-cash" flow. An invoice without a payment mechanism is a dead end; this plan makes invoices actionable. The PaymentProvider abstraction protects Phase 6 (SaaS) and future customers who may use a different payment processor.

## Output
- `src/lib/payments/provider.ts` — PaymentProvider interface
- `src/lib/payments/stripe-provider.ts` — Stripe implementation using Payment Links API
- `src/lib/actions/invoices.ts` — `sendPaymentLink` action added
- `src/lib/n8n.ts` — `notifyInvoicePaymentLink` helper added
- `src/app/dashboard/invoices/[id]/page.tsx` — "Send Payment Link" button replaces "Mark as Sent"; payment link URL displayed when present
- `.env.local.example` — STRIPE_SECRET_KEY placeholder added
</objective>

    **`src/lib/payments/provider.ts`** — plain TypeScript module (no 'use server'):

    ```ts
    export interface CreatePaymentLinkParams {
      invoiceId: string
      amountCents: number      // amount in cents (e.g. $12.50 → 1250)
      description: string      // shown on Stripe payment page
      metadata?: Record<string, string>
    }

    export interface PaymentLinkResult {
      url: string        // the hosted payment page URL
      externalId: string // Stripe payment link ID (e.g. plink_xxx)
    }

    export interface PaymentProvider {
      createPaymentLink(params: CreatePaymentLinkParams): Promise<PaymentLinkResult>
    }
    ```

    **`src/lib/payments/stripe-provider.ts`** — plain module (no 'use server'):

    Import `Stripe` from `'stripe'`.

    Export `class StripeProvider implements PaymentProvider`:

    Constructor: `constructor(private readonly secretKey: string)`

    `async createPaymentLink(params: CreatePaymentLinkParams): Promise<PaymentLinkResult>`:
    - Instantiate Stripe client: `new Stripe(this.secretKey)`
    - Call `stripe.paymentLinks.create({...})` with:
      ```ts
      {
        line_items: [{
          price_data: {
            currency: 'usd',
            product_data: {
              name: params.description,
            },
            unit_amount: params.amountCents,
          },
          quantity: 1,
        }],
        metadata: { invoice_id: params.invoiceId, ...params.metadata },
      }
      ```
    - Return `{ url: link.url!, externalId: link.id }`

    Export a factory function `getStripeProvider(): StripeProvider`:
    - Read `process.env.STRIPE_SECRET_KEY`
    - If missing or empty: throw `new Error('Stripe is not configured — set STRIPE_SECRET_KEY in environment')`
    - Return `new StripeProvider(process.env.STRIPE_SECRET_KEY)`

    Install: run `npm install stripe` in apps/print-shop-crm.

    **`src/lib/n8n.ts`** — append `notifyInvoicePaymentLink` helper:

    ```ts
    export async function notifyInvoicePaymentLink(
      invoiceId: string,
      customerEmail: string,
      customerName: string,
      paymentLinkUrl: string
    ): Promise<void> {
      await notifyN8n(process.env.N8N_WEBHOOK_INVOICE_PAYMENT_LINK, 'invoice.payment_link_sent', {
        invoiceId,
        customerEmail,
        customerName,
        paymentLinkUrl,
      })
    }
    ```

    **`src/lib/actions/invoices.ts`** — add `sendPaymentLink` export:

    Import `getStripeProvider` from `@/lib/payments/stripe-provider`.
    Import `notifyInvoicePaymentLink` from `@/lib/n8n`.

    `export async function sendPaymentLink(invoiceId: string)`:
    1. `await requireAdmin()` — reuse existing helper
    2. Fetch invoice with customer join:
       ```ts
       supabase.from('invoices')
         .select('id, amount, status, order_id, orders(customers(name, email))')
         .eq('id', invoiceId)
         .single()
       ```
    3. If not found: throw `'Invoice not found'`
    4. If `invoice.status === 'paid'`: throw `'Cannot send payment link for a paid invoice'`
    5. Extract `customer` with `Array.isArray()` guard on both joins
    6. Get Stripe provider: `const provider = getStripeProvider()` — will throw with clear message if key missing
    7. Create payment link:
       ```ts
       const result = await provider.createPaymentLink({
         invoiceId,
         amountCents: Math.round(Number(invoice.amount) * 100),
         description: `Invoice #${invoiceId.slice(0, 8).toUpperCase()}`,
         metadata: { order_id: invoice.order_id },
       })
       ```
    8. Update invoice:
       ```ts
       supabase.from('invoices').update({
         stripe_payment_link_url: result.url,
         stripe_payment_intent_id: result.externalId,
         status: 'sent',
       }).eq('id', invoiceId)
       ```
    9. Fire notification (fire-and-forget, non-blocking):
       ```ts
       await notifyInvoicePaymentLink(
         invoiceId,
         customer?.email ?? '',
         customer?.name ?? '',
         result.url
       )
       ```
    10. `revalidatePath('/dashboard/invoices/' + invoiceId)`
    11. `revalidatePath('/dashboard/orders/' + invoice.order_id)`
    12. `redirect('/dashboard/invoices/' + invoiceId)`
